A little bit about us

Who we are

We’re an open source project that was founded in 2023 (view the source code here). We connect you and other organizations so you can get the things you need, and get rid of the things you don’t. Not an organization? No big deal. You can browse our list of donatable items and organizations.

How can I help?

If you're a member of a charity organization or know any members, you can redirect them to us, where they can share their supplies. If you would like to donate supplies to charity organizations in need, you can do this by connecting with them via NPL too.

About the creator

Hi, I'm Oscar! I've been coding on and off since I was 12, and I've been coding consistently (almost every day) since I was 15. I'm fluent in Python, HTML/CSS/JS, Typescript, and SQL. The primary technologies I work with are Django, HTMX, the Django Rest Framework, and PostgreSQL. You can find more about me here.

Technical info for the nerds

This site is hosted on a VPS hosted by Digital Ocean (Nginx and Gunicorn, if you're familiar). The backend is Django and DRF, and the frontend is raw HTML/CSS/JS (check out the hellish JS I wrote because I refused to use a framework here), and we're using PostgreSQL as our database. You can access (and contribute to!) the source code here.
